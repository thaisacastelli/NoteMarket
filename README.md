# NoteMarket
Sample using Firebase authentication and pattern MVP. This application allows to manage shopping lists for supermarkets for example. Android 4.0 (API level 14) or higher support.

## About
This example can be opened in Android Studio and to run it you need to include the file "google-services.json" in the application's ```app/``` folder. From this file Firebase manages all the settings and credentials of your API. To download the "google-services.json" file, go to the Firebase Console (https://console.firebase.google.com) and import this example into a Firebase project using the "applicationId" value from the ```app/build.gradle``` file. Once created at Firebase console click on Settings icon in the left menu and select Project Settings, within these settings have an option to download the file "google-services.json" of the project.
